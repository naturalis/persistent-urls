Welcome to the biodiversitydata.nl PURL server. The PURL server provides persistent URLs for specimens from Naturalis, Xeno-Canto and Waarneming.nl.

PURLs take the following form:

Naturalis:

${baseUrl}/naturalis/specimen/{ObjectID}

Substitute {ObjectID} with an actual specimen ID. For example: ${baseUrl}/naturalis/specimen/U.1040749

Xeno-canto:

${baseUrl}/xeno-canto/observation/{ObjectID}

Substitute {ObjectID} with an actual observation ID from xeno-canto.org. For example: ${baseUrl}/xeno-canto/observation/XC401237

Waarneming.nl:

${baseUrl}/obsint/observation/{ObjectID}

Substitute {ObjectID} with an actual observation ID from waarneming.nl. For example: ${baseUrl}/obsint/observation/OBS.79206189
