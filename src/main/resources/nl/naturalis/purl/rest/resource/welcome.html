PURL server up and running!

Welcome to the Naturalis PURL server. Naturalis is in the process of creating persistent URLs (PURLs) for its main assets. For more information, see our documentation.

PURLs take the following form:

Naturalis:

http://data.biodiversitydata.nl/naturalis/specimen/{ObjectID}

Substitute {ObjectID} with an actual specimen ID. For example: http://data.biodiversitydata.nl/naturalis/specimen/U.1040749

Xeno-canto:

http://data.biodiversitydata.nl/xeno-canto/observation/{ObjectID}

Substitute {ObjectID} with an actual observation ID from xeno-canto.org. For example: http://data.biodiversitydata.nl/xeno-canto/observation/105075
